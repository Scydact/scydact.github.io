Generador de diagrama de flujo de efectivo

Commands:

  • t X: set the current time period to X.
  • tr X: "skip" X time steps (relative movement).
  • X: set a period to X.
  • m TEXT: sets a text (message) at this period.
  • ma TEXT: sets a text (message) at the previous period.
  • % TEXT: Comment, ignored by the parser.
  • h TEXT: Sets the graph's title.
  • mult X: Sets a global multiplier for the money

Multiple commands per line:

  • X X X: Sets level to A, B, and then C.
  • X X t Y X: Sets X, X on same t, changes to t=Y and then sets X on t=y.
  • Any text command (m, %, h...) will assume the rest of the line as the text.
